## Creating our first image


## about images

- this is a more like a blueprint for containers that need to created in order to run applications

## Notes

1) before we containarize an application we will need an image 
2) to create an image we will have to use a base image from docker hub and use it as a foundation
to build our own image
3 to build our own image we will have to create a dockerfile.
4) in this file we describe how our container should be like based off the image we will create

## steps 

1) open the project folder in your IDE (visual studio code)
2) create a file named (dockerfile) without any extension
3) once thats done we specify our instructions and save the file
4) open a terminal within vs code 
5) run docker build . (this will build an image)
