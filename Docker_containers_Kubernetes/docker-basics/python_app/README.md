#Random Number Generator (Pyth

This is a simple Python application that generates a random number between two values provided by the user.

The user is prompted to enter:
- a minimum number
- a maximum number

If the input is valid, the program generates a random number within that range. If the maximum value is less than the minimum, the program exits with an error message.

---

##Why I containerized this app

I containerized this application using Docker to better understand how even simple scripts can be packaged and run consistently across different environments.

This helped me practice:
- Creating a Dockerfile for a Python app
- Running interactive applications inside a container
- Understanding how user input works in containers

##What I learned

- How to containerize a Python application
- How Docker handles runtime execution
- The importance of environment consistency, even for small apps
