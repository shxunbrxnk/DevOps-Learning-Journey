## In this folder we are going to look at how to setup docker on windows


##prerequisites

- windows 10/11
- wsl 2
- hyper-v (virtualization)

## installation Steps

1) go to docker.com 
2) navigate to download and install
3) go through requirments and choose if you want to spin up in windows/linux/mac
4) check system type (windows+R msinfo32)
5) download installation file (may take a while)
6) install and follow the on screen prompts
7) Once done restart machine to run docker
8) After machine turns on you will be required to create a docker acc
9) once account created goto cmd and docker --help to check if it has been installed (if you get a list of recommendations then it has been successful)
10) install IDE (visual studio code)
